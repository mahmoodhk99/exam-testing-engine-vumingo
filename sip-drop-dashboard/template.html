📶 Nailer Drop Abuse Dashboard — sip:100@zain.com

Flags agents who drop the nailer from the softphone: a sip:100 drop with no matching In Job Break = abuse

The idea in one line

When an agent hangs up the sip:100@zain.com nailer call legitimately, POM logs an In Job Break. A nailer drop with no matching In Job Break = abuse — the agent hung up to dodge calls.

The table

  • Abuse drops = nailer drops − in-job breaks (per agent).
  • Abuse sessions = how many of the agent’s sessions had any abuse.
  • Abuse time = total off-work time caused by those drops (each drop until the next job they attached to).
  • Use search, the sort menu, and Only flagged agents to focus. Click a row to see each session and drop.

Colours

  • Red = abuse (a drop with no break).
  • Amber = dropped without a break, then went off before the next login.
